# Site Map and Layout Summary

## Global Shell
- Root layout applies fonts, providers, error boundary, and AI chat overlay.
- Public pages (login/register) render full-screen without sidebar/header.
- All other pages render with the left sidebar and top header.

## Navigation (Sidebar)
- Quick access: Home, Groups, Students, Timetable, Attendance, Reports
- Management: Assessments, Progress, POE Management, Compliance, Moderation
- Tools: Lesson Planner, Curriculum, AI Assistant, Settings
- Admin: User Management (admin-only)

## Layout Summary by Page
- Home: stats, quick actions, programme health, charts, activity, alerts, schedule
- Groups: group cards/list, rollout status, actions, exports
- Group Detail: rollout plan, projected completion, assessment status
- Students: filters, sort, attendance, compliance, bulk actions
- Assessments: per-unit tabs (Formative/Summative/Workplace), marking, summaries
- Assessment Checklist: group/module filters, per-student completion
- Attendance: mark daily attendance, bulk actions, alerts, history
- Progress: module and unit standard progress by student
- Compliance: attendance threshold compliance and report export
- Moderation: pending assessment moderation queue
- POE: physical evidence checklist by student
- Timetable: calendar and week schedule views
- Lessons: create lesson plans with module, group, date/time
- Curriculum: library view, AI upload, PDF export
- Curriculum Builder: search docs and build assessments
- Reports: generate daily report, PDF, AI text
- Settings: profile, notification, reminder, system, appearance, security
- AI Assistant: chat, semantic search, assessment generation
- Admin: admin overview and system cards
